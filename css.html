CSS (Cascading Style Sheets)

CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML. It dictates how elements should be rendered on screen, on paper, or in other media, allowing developers to control the layout, colors, fonts, and overall visual aesthetics of a web page.

How CSS Works

CSS works by applying styles to HTML elements. A CSS rule set consists of a selector and a declaration block: 

selector {
    property: value;
    property: value;
}
  • Selector: Points to the HTML element(s) you want to style (e.g., h1, p, .my-class, #my-id).
  • Declaration Block: Contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon.

Ways to Include CSS

There are three main ways to incorporate CSS into an HTML document:

  • Inline Styles: Applied directly to HTML elements using the style attribute (e.g., <p style="color: blue;">). Not recommended for large projects.
  • Internal (Embedded) Styles: Defined within a <style> tag in the HTML <head> section. Good for single-page styles.
  • External Stylesheets: The most common and recommended method. Styles are defined in a separate .css file and linked to the HTML document using the <link> tag in the <head> (e.g., <link rel="stylesheet" href="style.css">). This promotes reusability and maintainability.

Key CSS Concepts

  • Selectors: Element selectors (p), class selectors (.my-class), ID selectors (#my-id), attribute selectors, pseudo-classes (:hover), pseudo-elements (::before).
  • The Box Model: Every HTML element is considered a rectangular box, composed of content, padding, border, and margin. Understanding this is crucial for layout.
  • Layout: Techniques like Flexbox and CSS Grid are modern approaches for creating complex and responsive layouts.
  • Responsive Design: Using media queries to apply different styles based on screen size, allowing websites to adapt to various devices (desktops, tablets, mobile phones).
  • Cascading and Specificity: Rules that determine which styles are applied when multiple styles target the same element. Styles "cascade" from general to more specific rules.
