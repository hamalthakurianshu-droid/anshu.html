HTML (HyperText Markup Language)

HTML is the backbone of every web page. It's a markup language, not a programming language, used to structure content on the internet. It provides a standardized way to create headings, paragraphs, lists, images, links, and various other elements that form the skeleton of a web document.

Basic Structure of an HTML Document

Every HTML document follows a fundamental structure: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>
  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html lang="en">: The root element of an HTML page, specifying the document's language.
  • <head>: Contains meta-information about the HTML document (e.g., character set, viewport settings, page title, links to stylesheets, scripts).
  • <body>: Contains all the visible content of the web page.

Key HTML Elements

HTML uses various tags to define different types of content:

  • Headings: <h1> to <h6> for different levels of headings.
  • Paragraphs: <p> for blocks of text.
  • Links: <a href="url"> to create hyperlinks.
  • Images: <img src="path/to/image.jpg" alt="Description"> to embed images.
  • Lists: <ul> (unordered) and <ol> (ordered) with <li> for list items.
  • Divisions: <div> for grouping block-level elements.
  • Spans: <span> for grouping inline-level elements.
  • Forms: <form>, <input>, <label>, <button> for user input.

Attributes

HTML elements can have attributes that provide additional information about the element. Attributes are specified in the start tag and usually come in name/value pairs like name="value". Example: <a href="https://example.com" target="_blank">Visit Example</a>, where `href` and `target` are attributes.
